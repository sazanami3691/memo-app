# memo-app

GitHub Pagesで動かす、ローカル保存型のメモアプリです。

## 概要

このアプリは、ブラウザ上で動作するメモアプリです。

メモ本文、画像、手書きメモ、登録画像などのデータは、各端末のブラウザ内のIndexedDBに保存されます。

GitHub Pagesにはアプリ本体のみを公開し、個人的なメモ内容や画像データはアップロードしません。

## 主な機能

- フォルダ管理
- メモ作成、編集、削除
- テキストブロック
- RPGツクールMZ向け文章ブロック
- 画像ブロック
- 画像トリミング
- よく使う画像の登録
- 手書きブロック
- 検索
- メモ移動
- ピン留め
- JSONバックアップ書き出し、読み込み
- ライト / ダークテーマ切替

## 注意

以下のファイルはGitHubにアップロードしないでください。

- memo_backup_*.json
- backup*.json
- *.backup.json
- 個人的なメモ本文
- 画像入りバックアップJSON
- 手書き入りバックアップJSON

## デプロイメモ

GitHub Pagesのデプロイを再実行したい場合は、このREADME.mdに小さな変更を加えてコミットすると、新しいPagesデプロイが発生します。
